# Planilha interativa de investimentos no Excel
Este projeto é o desenvolvimento de uma planilha interativa de investimentos no Excel fruto do curso promovido da DIO em parceria com o banco Santander.

<a href="https://github.com/lucasdga1/Projeto_1_DIO_Excel/blob/main/Lab_1_Excel.xlsx">Planilha de Investimentos em FII</a>

## Objetivos e aplicabilidade
Esta planilha auxilia o investidor a escolher:

*  O perfil de investidor

* Percentual de investimento em cada tipo de FII

* Quantia de investimento em cada tipo de FII baseada no aporte mensal

* Quantia sugerida a ser aportada, baseada no salário do investidor

* Quantia de capital acumulado após tempo inserido pelo investidor, no caso de um investimento constante mensal

* Quantia de dividendos mensais baseados no capital acumulado

* Quantia de dividendos após períodos de tempo pré-determinados baseados no capital acumulado
